    • Responsive website

    First big project at KEA was responsive website. With a bit more then a month of coding I created this rather simple website. As I wanted to create a very simple website, I also wanted to include some basic design principles to make the website as esthetically pleasing as possible. First thing I tried to include in my final website is the golden ratio between burger navigation and logotype, so they are in perfect harmony. For the desktop site, I decided to make it in the horizontal symmetry. As the page is responsive, when the screen is smaller layout shifts accordingly in the mostly fluid pattern. I decided to use an asymmetrical structure for mobile and tablet layout.
